## OvsX

Ovvero la battaglia tra le O e le X


   O | X | O
  ------------
   O | O | X
  ------------
   X | X | X
  

  
Appunti:

Un agente arbitro (moderatore di partite)
Un agente controller (giocatore di partite)

Arbitro:
  - nuova partita
  - richiesta mossa
    - identificazione posizione pezzi
    - normalizzazione delle simmetrie
    - proposta mosse possibili
  - gestione archivio mosse
  - analizzatore fine partita
    - invio dati al giocatore

Giocatore:
  - scelta mossa
  - elaborazione risultato e modifica pesi
